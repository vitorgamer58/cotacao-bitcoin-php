# cotacao-bitcoin-php


Código simples em PHP que faz as solicitações para as exchanges de criptomoedas dos respectivos preços, e então calcula uma média ponderada pelo volume.

Exchanges:

```
Braziliex

Bitcoin Trade

Walltime

Bitcoin to You

Mercado Bitcoin

Novadax
```

Endereço: https://bitcao.notasdovitor.top/cotacao/
